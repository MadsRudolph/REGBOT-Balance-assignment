\section{Introduction}
\label{sec:introduction}

The REGBOT is a two-wheeled inverted pendulum. Without active control it falls; stable balance plus motion tracking requires a cascaded controller. This report documents four nested loops:
\begin{enumerate}[leftmargin=*,nosep]
    \item wheel-speed PI (innermost);
    \item balance PILead with post-integrator (Lecture~10 Method~2) -- stabilises the open-loop unstable pendulum;
    \item velocity PI -- drives the robot at a commanded speed while balancing;
    \item position P (outermost) -- moves the robot to a target distance.
\end{enumerate}
Each is designed analytically with the frequency-domain phase-balance method, verified in Simulink (Simscape Multibody), and validated on the physical robot.
